HydroGuard | Intelligent Water Management

HydroGuard is a simulated IIoT dashboard designed to demonstrate the application of AI and sensor analytics in large-scale building infrastructure.

Project Vision

This project explores how we can bridge the gap between static sanitary infrastructure and "Smart Building" systems. By applying real-time data visualization and anomaly detection, we can move from reactive maintenance to intelligent, proactive water management.

This simulation serves as a practical demonstration of my readiness for the Praxisintegriertes Bachelorstudium (PiBS) in Informatics/AI at Geberit International AG, showcasing my ability to integrate software intelligence with physical infrastructure requirements.

Key Engineering Features:

IIoT Monitoring: Real-time tracking of flow rates, pressure, and resource usage.

AI Anomaly Detection: Logic to automatically detect potential leaks (pressure drops) and escalate system status.

Operational Command Center: A simulated terminal interface for remote valve control and diagnostic execution.

Technical Stack

Frontend: HTML5, Tailwind CSS, JavaScript.

Concept: Industrial Internet of Things (IIoT) / Smart Infrastructure.

Developed as a demonstration of intelligent infrastructure design, focusing on sustainability and proactive maintenance for Geberit AG.
